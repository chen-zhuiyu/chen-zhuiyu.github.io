# 设计师作品集网站

一个现代、极简的作品集网站，用于展示设计作品，使用 HTML5、CSS3、JavaScript 和 GSAP 动画构建。

## 特性

- 简洁、极简的设计，聚焦于作品展示
- 适用于所有设备的响应式布局
- 使用 GSAP 实现的流畅动画和过渡效果
- 可筛选的作品画廊
- 详细的项目页面
- 包含工作和教育时间线的"关于"板块
- 联系表单
- 移动端友好的导航

## 使用技术

- HTML5
- CSS3（搭配 Tailwind CSS）
- JavaScript
- GSAP（GreenSock 动画平台）
- Font Awesome 图标

## 项目结构

```
/
├── index.html          # 主作品集页面
├── works/              # 项目详情页
│   ├── work1.html
│   └── ...
├── css/                # CSS 文件
│   └── style.css
├── js/                 # JavaScript 文件
│   └── script.js
└── assets/             # 图片和其他资源
    └── images/
```

## 使用方法

1. 克隆或下载仓库
2. 在浏览器中打开 `index.html`
3. 要查看项目详情，点击画廊中的任意作品项

## 自定义

- 将 `assets/images/` 中的图片替换为你自己的作品
- 更新 `index.html` 和作品详情页中的文本内容
- 在 Tailwind CSS 配置中修改配色方案
- 在 JavaScript 文件中调整动画效果
